### Setup
1. Make sure you have NodeJS installed on your machine. If not, you can download it [here](https://nodejs.org/en/download/) or install it using [Homebrew](https://brew.sh/).
2. Make sure you have TypeScript installed on your machine. If not, you can install it using `npm install -g typescript`.
3. Clone this repository to your local machine.
4. Run `npm install` to install all the dependencies.
5. Run `npm prisma:generate` to generate the Prisma from the schema.
6. Run `npm run dev` to start the development server.
